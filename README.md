### ✨ Setup Environment

```bash
$ pnpm install
$ pnpm run prepare
$ cp .env.example .env
```

### ✨ Quick Start in `Docker`

```bash
$ docker-compose up --build
```

### ✨ Quick Start in `local`

```bash
$ bin/dev
```

### ✨ Running unit test

```bash
$ bin/test
```

### ✨ Running Python shell

```bash
$ bin/console
```

### ✨ Linting

```bash
$ bin/lint
```
